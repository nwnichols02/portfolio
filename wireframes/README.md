# Portfolio wireframes

## Desktop one-page scroll (1440px artboard)

**File:** `portfolio-desktop-wireframe.excalidraw`

Open in **Excalidraw** (desktop app or [excalidraw.com](https://excalidraw.com)):

- **Desktop:** Excalidraw Desktop → File → Open → select `portfolio-desktop-wireframe.excalidraw`
- **Web:** [excalidraw.com](https://excalidraw.com) → Menu (⋮) → Open → choose this file

### Layout

- **Artboard:** 1440px wide × 4200px tall
- **Margins:** 80px left/right (12-column grid)
- **Sections (top to bottom):**
  1. **Header** (fixed) – 80px – wordmark, subtitle, nav (Home | About | Projects | Skills | Experience | Contact), theme + glyph toggles
  2. **Hero** – ~750px – two-column: headline, subhead, morph hint, 3 metric cards, CTAs (View Projects, Download Resume), welding-bead progress; right: hero visual card (welder silhouette, waveform, map grid), ham radio dial
  3. **About** – ~450px – “About” label, title, bio, “What I optimize for” bullets (2 columns); right: Call Sign Panel (NN0DEV, QTH, bands)
  4. **Skills** – ~550px – 3 cards on steel-plate background: Front End, Back End, Architecture & Extras (chip rows as specified)
  5. **Projects** – ~700px – “Projects” title, frequency dial (Front End | Architecture | Play), 3 project cards (Rose Creek YSA, Tesla Clone, Mini Games), project detail modal overlay hint
  6. **Experience** – ~650px – center timeline spine with alternating nodes (UtiliSource, Farmers Insurance)
  7. **Education** – ~300px – 2 cards (BloomTech, Northern Wyoming Community College)
  8. **Contact** – ~600px – “Let’s build something reliable”, form (Name, Email, Message), Send Signal button, social icons; right: Signal Map with ping markers
  9. **Footer** – thin strip with © Nathan Nichols – Engineered with React, TypeScript & Vite

Visual notes in the wireframe: dark header bar, metric/stat cards, welding-bead progress, hero image placeholder, Call Sign Panel with arc-weld accent, weld dots on skill cards, timeline as “cable” with node circles, map contour hints, Signal Map pings.
